# **App Name**: Smart Shopper

## Core Features:

- Add Item: Add new grocery items to the shopping list with an 'Add Item' button and auto-record purchase date.
- Remove Item: Remove items from the grocery list with a 'Remove' option for each item.
- View Grocery List: Display the current grocery list in a table or list format on the page.
- Suggest Re-Purchase: Suggest items to re-purchase based on the user's purchase history tool.
- Suggest Healthier Alternatives: Recommend healthier alternatives for items in the grocery list based on a predefined dictionary tool.
- Check Expiring Items: Notify the user if any items in their purchase history are about to expire or have already expired based on stored expiry times.
- Local Storage: Persist the current list and purchase history across sessions using localStorage.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to evoke a sense of calm and organization.
- Background color: Light gray (#F5F5F5), offering a clean and neutral backdrop.
- Accent color: Muted green (#A9DFBF) for a fresh and healthy feel, used for important notifications and actions.
- Body and headline font: 'PT Sans', a humanist sans-serif, for clear readability and a touch of warmth.
- Simple, outline-style icons to represent grocery items and actions. For example, a shopping cart icon for adding items, a trash can icon for removing items.
- Clean and simple layout with a focus on readability and ease of use. The grocery list should be prominently displayed, and action buttons should be easily accessible.
- Subtle transitions and animations when adding, removing, or updating items in the grocery list. Use a simple fade-in or slide-in effect.
